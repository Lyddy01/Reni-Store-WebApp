##  Reni Store Web App (UI Clone Project)

This project is a front-end clone of an existing e-commerce website, built using HTML, CSS, and JavaScript.  
It focuses on replicating the original design while implementing smooth UI interactions, including quote transition animations.

##  Live Demo
 https://lovereni.netlify.app/

## Features
- Responsive layout for mobile and desktop
- UI design inspired by a real e-commerce website
- Smooth quote transition animations

## Technologies Used
- HTML5
- CSS3
